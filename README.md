# Burnjs
burnjs是系列教程《使用Typescript封装一款基础企业级Web框架》配套框架，全部使用Typescript编写完成。

# 快速开始
```bash
#安装脚手架
npm install -g burn-init
#初始化项目
burn-init -init-ts myapp
#进入目录
cd myapp
#安装依赖
npm install
#启动项目
npm run dev
```
